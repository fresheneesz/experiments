-- Instalation --

1. install node-v0.10.3-x64

2. Create folder hierarchy:
  A. create an unversioned project folder called "node"
  B. pull the newFrame folder from SVN into the "node" folder

3. run node node/newFrame/install.js

4. fill in your configuration file at node/configuration.js


-- Installing a new module --

1. add the module name to the list of modules in node/newFrame/install.js
2. rerun node/newFrame/install.js
